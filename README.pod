=head1 Group::Git::Taggers::Perl

This package is a tagger from L<Group::Git> it will tag any repository that
looks like a Perl repository (ie has a Perl Makefile.PL, Build.PL or dist.ini
file).

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
